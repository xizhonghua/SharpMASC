Origami Viewer
==============
Version 0.1, 3/9/2014

Requirement:

.NET/Mono 4.0+

Usage
-----
Windows : run.bat
Linux/Unix : run.sh

Commandline:
[mono] Origami.exe -c models.xml


Shortcuts
---------
-- Origami --
?     : Display this message
n     : Next model
p     : Previous model
-- Folding --
space : Toggle Animation
,     : Fold
.     : Unfold
[     : Fold (one step)
]     : Unfold (one step)
t     : Folding to goal state
r     : Folding to initial state
-- Display --
a     : Toggle showing assistant creases
-- Dumping --
M     : Dump current state to obj file
